# Audio Fingerprinter

A few sample databases to import can be downloaded here: https://s3.amazonaws.com/writecomments.com/librarymusicthemes/index.html.

## How to use

<img src="https://user-images.githubusercontent.com/6477175/158115413-28a9a935-cbe2-4f39-b680-c46b9b080401.png" width="800" />

- Select a directory with audio files for fingerprinting and store the created fingerprint database.
- Filter directory contents by filename substrings (e.g. extensions).
- Adjust max CPU usage.
- The About window shows the installed Python version.

## How to install

Download the latest release from https://github.com/mitin001/audfprint-gui/releases (.exe for Windows, .dmg for macOS).

### Windows

<img src="https://user-images.githubusercontent.com/6477175/158116934-df4494c5-f8fc-4ebd-8e78-a6a394e7527a.png" width="600" />

<img src="https://user-images.githubusercontent.com/6477175/158116778-a9b996a7-dd59-419b-b018-7817c9daf07f.png" width="400" />

If SmartScreen does not recognize the installation file downloaded from here, right-click on it, and select properties. Under Security, select Unblock.

<img src="https://user-images.githubusercontent.com/6477175/158115720-1c4348d2-c260-4a90-af20-5018fe6814f4.png" width="500" />

If Python is not installed, Audio Fingerprinter will take the user to its download page.

![image](https://user-images.githubusercontent.com/6477175/158115850-0480720d-1789-4682-96c4-3b56ebfadbfb.png)

On the first run, Audio Fingerprinter will install all of its dependencies.

### MacOS

<img src="https://user-images.githubusercontent.com/6477175/159143253-32c187a0-0180-4567-91a8-7ecb13a85765.png" width="400" />

On MacOS, you will get a warning that the developer cannot be verified.

<img src="https://user-images.githubusercontent.com/6477175/159143270-ea580ec7-eac3-4465-aac5-0153b8cb03d2.png" width="500" />

To get around the error, go to System Preferences / Security & Privacy.

<img src="https://user-images.githubusercontent.com/6477175/159143273-3b1d2c00-3749-49f5-885c-b4b68b63b285.png" width="600" />

Click Open Anyway for audfprint-gui.
